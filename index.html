Hired

Server-side project built using Python/Django & React

This is a simple job finding application. Where a user can log on and find some one to fulfill a role in their life. Let's say a user needs a sink repaired, or a TV fixed, heck, a user can find a guitarist for a session or someone to write a bio or a build a website. Any job can be found locally through Hired. I used Python/Django to build an API using ORMs and a little of direct Sqlite3 queries. Then paired it with a React application. I als incorpated Semantic UI react for minor styling and a ton of hard coded CSS3 as well. Enjoy the app and let me know if you have any questions!
